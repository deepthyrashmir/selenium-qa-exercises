# selenium-qa-exercises
Selenium automation exercises web tables and dynamic loading scenarios.
# Selenium QA Exercise

## Tools Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Chrome (Headless)

## Task 1: Web Tables
- Count table rows
- Add a new user
- Verify row count increase
- Edit user age
- Validate updated data

## Task 2: Dynamic Loading
- Click Start button
- Wait for dynamic content
- Verify "Hello World!" text
- Run in headless mode
- Capture screenshot

## How to Run
- Clone the repo
- Run: `mvn test`
